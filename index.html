⚠️ Brauzer Xavfsizlik Bloki (CORS / file://)

Siz o'yin fayllarini brauzerda to'g'ridan-to'g'ri (double click qilib) ochgansiz. Brauzer xavfsizlik tizimi (CORS) tufayli JavaScript fayllari yuklanishdan to'xtatildi va tugmalar ishlamaydi.

O'yinni ishga tushirish uchun quyidagi usullardan birini ishlating:

1. Terminalda npm run dev buyrug'ini bering va ekranda chiqqan http://localhost:5173/ havolasini oching.

2. Loyihani GitHub Pages-ga yuklang (u yerda onlayn tarzda ishlaydi).
